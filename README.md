## 📌 About this project

pdfstudio acts as a local toolkit for your PDF documents. It runs inside your web browser. This software handles common tasks such as locking, unlocking, merging, splitting, and rotating PDF files. Because the code runs on your machine using WebAssembly, your files never travel to a server. Your data stays on your device at all times.

## 🎛️ How to use the tools

The application provides a visual interface for all PDF actions. Click the button that matches your task to begin.

### Lock and Unlock Files
Use these tools to protect sensitive documents. You can add a password to restrict access or remove a password from a protected file. Select your file, enter the password, and click the process button.

### Merge PDF Documents
Combine multiple files into one document. Drag your files into the main window. Arrange the files in the order you want. Click the merge button to create your final file.

### Split and Rotate
Split a single large file into individual pages. Rotation tools allow you to change the orientation of pages that appear sideways or upside down. Choose the rotation degree or specify page ranges to split.

## 🔒 Privacy and Security

Data security takes priority. Traditional web services upload your documents to a cloud server. This software avoids that process entirely. The browser environment handles all tasks locally. No external entity monitors your files or stores your data. The engine uses WebAssembly to keep performance fast while maintaining high standards for personal privacy.
